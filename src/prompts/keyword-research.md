# Keyword Research Prompt

You are researching Google Ads keywords for Shopping Event VIP, a Belgian fashion outlet. Target markets: Belgium (Dutch + French) and Netherlands (Dutch).

## Keyword Strategy

### Match Type Guidelines
- **Exact match** `[keyword]`: Brand names, high-intent purchase terms
- **Phrase match** `"keyword"`: Specific product categories + modifiers
- **Broad match**: Only with Smart Bidding (tCPA/tROAS) for discovery

### Keyword Grouping (SKAG-light approach)
Group keywords by tight themes (3-5 keywords per ad group):
- Each ad group = one intent/theme
- Include NL and FR variants in separate campaigns (language targeting)

### Dutch Keywords (NL + BE)
Core terms:
- merkkleding outlet, designer kleding outlet, merkkleding sale
- [brand] outlet, [brand] korting, [brand] sale
- [category] outlet (bijv. "jassen outlet", "schoenen outlet")
- outletwinkel online, merkkleding online kopen
- goedkope merkkleding, merkkleding met korting

Modifiers:
- sale, korting, aanbieding, actie, outlet, goedkoop, online
- heren, dames, kinderen
- winter, zomer, lente, herfst

### French Keywords (BE Wallonia + Brussels)
Core terms:
- vêtements de marque outlet, mode designer outlet
- [brand] outlet, [brand] soldes, [brand] pas cher
- [category] outlet (ex: "vestes outlet", "chaussures outlet")
- magasin outlet en ligne, acheter vêtements de marque
- vêtements de marque pas cher, mode à prix réduit

Modifiers:
- soldes, réduction, promotion, outlet, pas cher, en ligne
- homme, femme, enfant
- hiver, été, printemps, automne

### Negative Keywords (standard list)
- gratis, free, gratuit
- tweedehands, occasion, seconde main
- nep, fake, contrefaçon
- vacature, emploi, job
- klacht, avis négatif, complaint

## Output Format
For each keyword group, provide:
1. Ad group name
2. Keywords with match types
3. Suggested NL and FR ad copy themes
4. Estimated search volume tier (high/medium/low)
